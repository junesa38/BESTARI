Content Management System

Buat blog post atau tool baru untuk website BESTARI

Cara Penggunaan:

  • Isi form di bawah untuk membuat konten baru
  • Pilih tipe konten: Blog Post atau Tool
  • Klik "Generate" untuk membuat file
  • Copy kode yang dihasilkan dan simpan sebagai file .html atau .json
  • Upload file ke folder yang sesuai di GitHub repository
